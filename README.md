# Phishing-Simulation- Awareness and Reporting
# Overview
This project documents a live phishing simulation conducted to evaluate employee vigilance against credential-harvesting attacks following a phishing-awareness training program.
# Purpose
The simulation was designed to measure behavioral changes and security awareness by focusing on:
Lowering link-click frequency,
Reducing credential submission attempts,
Improving incident reporting.
# Outcome
Results from the exercise help organizations assess the effectiveness of phishing training, identify residual risk, and strengthen human-centric security controls.
# Disclaimer
This simulation was conducted strictly for educational and security-awareness purposes and did not involve real credential compromise.
